wof/svg: font properties like weight and style(italic) are 'overridden' by the font data. Specifying has no effect

google fonts: the fonts are looked up by specifying the family, style, and weight as a 'lookup'. google provides the css below, and if you you provide the proper parameters, you find the font. below is the sheet that google provides as a way to access these fonts:

@font-face {
  font-family: 'Orbitron';
  font-style: normal;
  font-weight: 400;
  src: local('Orbitron-Light'), local('Orbitron-Regular'), url(https://fonts.gstatic.com/s/orbitron/v7/HmnHiRzvcnQr8CjBje6GQnYhjbSpvc47ee6xR_80Hnw.woff2) format('woff2');
  unicode-range: U+0000-00FF, U+0131, U+0152-0153, U+02C6, U+02DA, U+02DC, U+2000-206F, U+2074, U+20AC, U+2212, U+2215, U+E0FF, U+EFFD, U+F000;
}
